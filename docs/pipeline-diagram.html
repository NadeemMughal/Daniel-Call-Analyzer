Stage Breakdown

Entry — Fathom Webhook
  • Fathom fires POST when any call recording ends
  • HMAC-SHA256 signature verified against secret
  • Duplicate check via source_id lookup
  • If duplicate → silent 200, stop processing
Ingest
  • New row in calls table, status: pending
  • Stores transcript_raw, transcript_segments, audio_url
  • Records duration, recorded_at, source_id
Classify — Groq Llama 3.1 8B
  • First 2,000 chars sent to fast 8B model
  • Returns: discovery / ads_intro / launch / follow_up / team / other
  • Updates calls row, status: processing
  • Routes to team branch OR sales branch
Rule Engine — Pure JavaScript
  • Banned words — e.g. "mate" on discovery calls → critical finding
  • Filler words — "um" >5 times → warning finding
  • Talk ratio — rep >60% of call → warning finding
  • All findings stored with coaching suggestion text
  • No LLM — deterministic, instant, zero cost
LLM Scorecard — Groq Llama 3.3 70B
  • Full transcript + active rubric sent to 70B model
  • Scores 5 criteria: talk ratio, question stack, pain surfacing, objection handling, solution timing
  • Weighted overall score 0–10
  • Every strength/improvement must include exact transcript quote
  • Rubric is editable from the portal — no code changes needed
Meeting Summary — Groq Llama 3.3 70B
  • Only runs for team/internal calls
  • Extracts: title, key points, decisions made (with owner)
  • Action items with owner name, due date, priority
  • Open questions + next meeting topics
  • Stored as scorecard with overall_score = null
Notify — Email per Participant
  • Call marked scored in database
  • Fetches all internal participants with emails
  • One personalised HTML email per rep
  • Contains: score, summary, top 3 strengths, top 3 improvements
  • Deep-link button to call detail page in portal
Database Tables Written
  • calls — created on ingest, status updated each stage
  • rule_findings — banned words, fillers, talk ratio
  • scorecards — overall score, summary, strengths, improvements
  • scorecard_evidence — exact quotes with timestamps
